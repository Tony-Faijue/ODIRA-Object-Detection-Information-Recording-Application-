ABOUT

What is Objet Detection?

Object detection is a computer vision technique that locates objects in digital images. It consists of training computers to see as humans do, by recognizing and classifying objects to semantic categories. Object localization is a technique used to determine the location of specific objects by drawing a bounding box.

What is ODIRA?

ODIRA stands for Object Detection Information Recording Application. ODIRA uses the MobileNet SSD v3 detection model that is trained on the COCO dataset. This application allows users to upload an image or take a picture with their webcam. Users can adjust the settings parameters before processing to detect specific category of objects and apply stricter or looser filtering. Once the image has been processed the application returns information about what objects were detected, how many per class and a total count of objects detected.

How to Use ODIRA?

To use ODIRA, the user must either use the webcam to take a picture or upload an existing image. The user can then adjust the settings parameters. Object Threshold parameter determines the model's certainty. (Higher values for High certainty and Lower Value for Low certainty). NMS (Non-Maximum Supression Threshold) parameter determines how boundary boxes of objects should overlap. (Higher values for more overlap and Lower values for less overlap). Once the image has been submitted the user can click the process button to process the image. The results will be displayed in the Results sections.
